Реализованы проекты на python по машинному обучению с отчётами в формате .ipynb или HTML по следующим темам:
- Линейные модели
- Метод ближайших соседей
- Решающие деревья
- Кластеризация
- Частичное обучение
- Градиентный бустинг

### Основные технологии
- Python (numpy, pandas, sklearn)
- Jupyter notebook
